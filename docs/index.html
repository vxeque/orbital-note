Arquitectura Modular: Cómo Escalamos Orbital Note

3 de Marzo, 2026 Arquitectura Técnico

Uno de nuestros mayores logros en el desarrollo de Orbital Note fue crear una arquitectura escalable y mantenible. Utilizamos un enfoque modular basado en características que separa claramente las responsabilidades.

Nuestra arquitectura consta de varias capas:

  • Router - Expo Router para navegación
  • Presentación - Componentes React Native
  • Estado - React Context API
  • Lógica de Negocio - Servicios especializados
  • Datos - AsyncStorage + Google Drive API

Sincronización Google Drive: Cómo Funciona

1 de Marzo, 2026 Backend Tutorial

La sincronización con Google Drive es el corazón de Orbital Note. Permite a los usuarios acceder a sus notas desde cualquier dispositivo, en cualquier momento.

El sistema funciona así:

  1. El usuario crea/edita una nota localmente
  2. La nota se guarda en AsyncStorage del dispositivo
  3. Se dispara un job de sincronización en background
  4. Se autentica con Google usando el token guardado
  5. Se carga/actualiza la nota en Google Drive
  6. Se resuelven conflictos (última escritura gana)
