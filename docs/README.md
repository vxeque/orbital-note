# Orbital Note

Orbital Note is a modern, lightweight note-taking application focused on simplicity, speed, and flexibility.
The goal of the project is to provide a clean environment for writing, organizing, and managing notes while remaining open source and accessible to developers.

---

## Features

* Simple and fast note creation
* Cross-platform support (web and mobile)
* Dark and light mode
* Lightweight and responsive interface
* Privacy-focused design
* Multi-language support
* Modular architecture for future extensions

---

## Tech Stack

Orbital Note is built using modern development tools:

* React Native
* Expo
* JavaScript / TypeScript
* HTML and CSS
* Node.js for tooling and development

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/vxeque/orbital-note.git
cd orbital-note
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npx expo start
```

After running the development server, you can open the application using:

* Expo Go on a mobile device
* An Android emulator
* An iOS simulator
* A web browser

---

## Documentation

The project includes a documentation website located in the `docs/` directory.

To open it locally:

```
cd docs
python -m http.server 8000
```

Then open:

```
http://localhost:8000
```

The documentation includes:

* Project overview
* System architecture
* Setup instructions
* Contribution guidelines
* Security information
